# HYU-AISW

한양대학교와 함께하는 2026 전국 청소년 SW AI 경진대회 팀 프로젝트 저장소입니다.

## 개발 환경

- React
- TypeScript
- Vite
- Oxlint

## 실행

```bash
npm install
npm run dev
```

## 검증

```bash
npm run lint
npm run build
```

## 대회 일정

- 2026-07-18 14:00: 개회식 및 주제 공개
- 2026-07-18 14:30: 산출물 개발 시작
- 2026-07-19 10:00: 산출물 개발 마감
- 2026-07-19 10:00-10:30: 구글폼 제출

## 작업 원칙

- 주제 공개 전에는 공통 골격과 제출 흐름만 준비합니다.
- Zoom 링크, 참가코드, 개인 연락처 같은 민감한 정보는 저장소에 기록하지 않습니다.
- 기능은 MVP 먼저 만들고, 디자인과 발표 자료는 작동 확인 후 다듬습니다.
- 제출 전에는 빌드 성공, 시연 흐름, 산출물 링크, 팀원 정보가 모두 맞는지 확인합니다.
